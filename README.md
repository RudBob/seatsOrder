# seatsOrder
这是一个作为预约系统，使用的技术为：
编程语言：Java语言，
框架：SpringBoot + Mybatis。
数据库：MySQL+ Redis

运行本项目只需要先运行项目中的sql文件后，更改application.yml中的数据库地址和redis地址后即可。

Java代码行数大约在1500行左右(因为使用了lombok，所以更显单薄)，逻辑十分简单。
